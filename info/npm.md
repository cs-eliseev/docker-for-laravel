NPM
====

NPM is a package manager for the JavaScript programming language.

### NPM install

1. Install Node.js
    ```shell
    apt install nodejs
    ```

1. Install NPM
    ```shell
    apt install npm
    ```

1. Check version
    ```shell
    nodejs -v
    ```
    
### NPM command

1. NPM init project
    ```shell
    npm init
    ```

1. NPM install all package
    ```shell
    npm install
    ```

1. Ðžnly install entire projects at a time
    ```shell
    npm ci
    ```
